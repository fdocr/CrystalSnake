# crystal-battle-snake-starter

A starter project for [Battle Snake](https://play.battlesnake.com/) written in Crystal

## Installation

The app uses [sam.cr](https://github.com/imdrasil/sam.cr) for ease of development.


```bash
shards install
make sam dev
```
